# parceljs 

## 설치 

```bash
npm install -g parcel-bundler
```

## 시작하기 

- index.html
    ```html
    <html>
    <body>
    <script src="./index.js"></script>
    </body>
    </html>
    ```
- index.js
    ```js
    console.log('Hello');
    ```

- 내장용 서버로 실행
    ```bash
    parcel index.html
    ```
    > localhost:1234 로 접속 하면 된다. 
    > 파일변화를 감지하고 리로딩 된다. 
    
- 만약 본인 소유의 서버가 있다면 watch 모드로 구동할 수 있다. 
    ```bash
    parcel watch index.html
    ```
